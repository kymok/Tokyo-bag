# Tokyo-bag

## About
This is a pattern for shoulder bag which can hold followings:

- An A4-sized writing pad
- An iPad Pro or a book
- A pencil case

## Materials

- Fabric
  - Outer fabric
  - Fusible interlining for outer fabric
  - Lining fabric
- Fittings
  - 2x Square ring 
  - 1x Slider bar buckle
  - 1x Magnetic fastener
